# ARK Architecture Specifications

This directory holds detailed architecture specifications for major ARK capability
domains and work contours.

**Important:**

- Documents here may be **approved but future-facing** — inclusion does **not** mean the
  runtime is implemented.
- `docs/PHASE_CONTRACTS.md` is authoritative for **implementation status**.
- `docs/PRODUCT_VISION_2026.md` owns the high-level **phase map**.
- Implementation happens incrementally through separately approved phases; a specification
  being present here is a direction, not a claim of working code.
